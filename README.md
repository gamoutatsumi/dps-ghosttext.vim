# dps-ghosttext.vim

[GhostText](https://ghosttext.fregante.com) plugin for denops.vim

## Requirements

- [denops.vim](https://github.com/vim-denops/denops.vim) is installed.
- Deno 1.13.0 or later

## Usage

In default, autostart GhostText server.

Manually: `:GhostStart`
